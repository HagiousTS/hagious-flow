Este pacote contém 6 telas + redirect index, todas autocontidas
(theme.css e theme.js inlinados em cada arquivo).

Arquivos:
  index.html                ← redireciona automaticamente pro dashboard
  hagious-flow.html         ← Dashboard executivo (PORTA DE ENTRADA)
  hagious-ia-coo.html       ← IA COO em fullscreen
  hagious-projeto.html      ← Projeto detalhe (Implantação Ômega)
  hagious-inbox.html        ← Inbox de Triagem de OS
  hagious-capacity.html     ← Capacity Planner
  hagious-foco.html         ← Modo Foco do consultor

═══════════════════════════════════════════════════════════════
  COMO USAR
═══════════════════════════════════════════════════════════════

LOCAL (offline):
  1. Descompacte o ZIP
  2. Duplo-clique em "index.html"
  3. Use a sidebar pra navegar entre telas
  4. Tema: ícone de paleta no canto INFERIOR direito

PUBLICAR ONLINE (sem servidor):
  1. https://app.netlify.com/drop
  2. Arrasta a pasta descompactada
  3. URL pública gerada em 10s

═══════════════════════════════════════════════════════════════
  TROCA DE TEMA
═══════════════════════════════════════════════════════════════

Botão de paleta no canto INFERIOR DIREITO.
  • Clique 1x → expande as 3 opções de tema
  • Clique novamente fora ou Esc → fecha

  🟢 Sankhya Light    → padrão claro
  🌑 Sankhya Dark     → escuro corporativo
  🟣 Hagious Purple   → tema original

Preferência salva em localStorage.
